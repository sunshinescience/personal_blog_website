Creating a diagonal two color background

Feb 24, 2020

Example black and grey diagonal background.

This is a quick way to create a two-color diagonal background, with an optional (additional) line at the same angle in the background (see image above).

My previous post is still a work in progress to highlight similarities and differences between CSS Grid and CSS Flexbox.

I initially used a background image to make a two-color diagonal background. Then I decided it might work better to be able to change the angle and colors in the CSS itself. One can use several different backgrounds together; just separate them with a comma. Here is an example of what the code for the above image is:

body {
background:
linear-gradient(-20deg, transparent calc(45% - 4px), rgba(0,0,0,1) calc(45% - 4px), rgba(0,0,0,1) calc(45% + 4px), transparent calc(45% + 4px)),
linear-gradient(-20deg, grey 50%, black 50%);
}

The first linear gradient makes a diagonal line. The rotation can be changed with degree angle. The vertical position can be changed with the calc (%).The second linear gradient makes a diagonal two color background at various angles. If you want to set a certain height, use min-height. The colors can be changed as well.
