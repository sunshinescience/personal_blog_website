CSS Grid & CSS Flexbox

Flexbox

Flexbox is a CSS Layout Module that makes it easy to design a flexible responsive layout structure.

To get started, define a flex container.

A Flexible Layout needs to have a parent element with the display property set to flex.

Direct child elements of the flexible container automatically become flexible items.
