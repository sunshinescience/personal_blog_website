CSS Grid & CSS Flexbox

Image gallery

I made an image gallery and tried a few things to get items to lign up horizontally. I also wanted them in a specific order. It seems that there are a few ways to get images to line up side by side:

  • 1.) Use float:left but note that you have to clear the float using overflow:hidden on parent element.
  • 2.) Use white-space: nowrap on the parent element and use display: inlineon the child element.
  • 3.) Set a background-image for the element and use CSS grid.

The first two didn't work in this instance. An example of the third is as follows:

CSS

/* Designating images in the image gallery */
#one{
background-image: url("../images/image1.jpg");
background-size: cover;
border: 5px solid black;
}

#two{
background-image: url("../images/image2.jpg");
background-size: cover;
border: 5px solid black;
}

#three{
background-image: url("../images/image3.jpg");
background-size: cover;
border: 5px solid black;
}

.photo-gallery{
width: 1fr;
height: 200px;
padding-bottom: 0;
}

.img-example{
padding-left: auto;
padding-right: auto;
}

.nested-img-example{
display: grid;
grid-template-columns: 1fr 1fr 1fr;
grid-template-rows: 1fr;
}

HTML

<div class="img-example">
<div class="nested-img-example">
<div id="one" class="photo-gallery">
<div id="two" class="photo-gallery">
<div id="three" class="photo-gallery">
</div>
</div>


The third worked (see above images), but I still wanted to be more flexible.

...enter Flexbox.

Flexbox

Flexbox is a CSS Layout Module that makes it easy to design a flexible responsive layout structure, without using float or positioning.

Flex Container

To get started, define a flex container.

.container {

display: flex; /* or inline-flex */

}

A Flexible Layout needs to have a parent element with the display property set to flex.

Then, direct child elements of the flexible container are flexible items.

Note that in the below image gallery, the flex container is shown in blue, whereas the flex items are the images. A container with flex-wrap: nowrap; will never wrap its items. In the image gallery below, the following code is used:

CSS

.flex-container {
display: flex;
flex-wrap: nowrap;
background-color: DodgerBlue;
height: 180px;
}

.flex-container > div {
min-width: 180px;
margin: 5px;
object-fit: contain;
}

HTML

<div class="flex-container">
<div> <img src="../images/HydePark1.jpg" width="240" height="180"> </div>
<div> <img src="../images/ZenRocks1.jpg" width="240" height="180"> </div>
<div> <img src="../images/Zion1.jpg" width="240" height="180"> </div>
</div>

The flex container properties are:

  • flex-direction - The direction of the flexible items inside a flex container
  • flex-wrap - Whether or not flex items should wrap
  • flex-flow - Shorthand property for flex-direction and flex-wrap
  • justify-content - Horizontally aligns the flex items
  • align-items - Vertically aligns the flex items
  • align-content - Modifies the behavior of the flex-wrap property. It is similar to align-items, but it aligns flex lines

For more information see the documentation. Additionally, for more information about Flex Container, see here
