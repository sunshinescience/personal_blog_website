Users can create a custom playlist and save it to their Spotify account.

Creating the React App

As a Codecademy project, I created a React app, which can be found at http://jamjammm.surge.sh. This involved using React components, passing state and requests with the Spotify API. This web app allows users to search the Spotify library by entering a song, album or artist. They can then create a custom playlist and save it to their Spotify account. The app will initially prompt the user to login to their Spotify account, and then they can then create their playlist and add it to their Spotify account.
